# 🎓 Student Management System

A simple **Student Management System** built using **Flask** and **PostgreSQL** that demonstrates complete CRUD (Create, Read, Update, Delete) operations with a clean and responsive user interface.

## 🚀 Features

- ➕ Add Student
- 📋 View Student Records
- ✏️ Update Student Details
- 🗑️ Delete Student
- 📊 Dashboard (Total Students & Departments)
- 🔍 Search Students
- 🏢 Filter by Department
- 📱 Responsive UI

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Flask (Python)
- **Database:** PostgreSQL
- **Database Driver:** psycopg2

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/student-management.git
```

2. Navigate to the project folder

```bash
cd student-management
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Configure PostgreSQL and update your database credentials in `database.py`.

5. Run the application

```bash
python app.py
```

6. Open your browser

```
http://127.0.0.1:5000
```
